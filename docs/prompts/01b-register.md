A clean, trustworthy authentication page for a sports management platform, featuring a centered form for new user registration.

**DESIGN SYSTEM (REQUIRED):**

- Platform: Mobile-first (Ionic App)
- Theme: Clean, minimal, institutional financial style
- Background: Pure White (#ffffff)
- Surface: Soft Gray (#f9fafb) for the form card
- Primary Accent: Blue (#0052ff) used SCARCELY for the primary submit button only
- Text Primary: Ink Black (#0a0b0d) for headings
- Text Secondary: Muted Gray (#7c828a) for labels and placeholders
- Typography: Sans-serif (Inter), regular weights (400) for headings
- Buttons: Pill-shaped (100px radius), full-width on the form
- Cards: Softly rounded (24px radius), flat with no decorative shadows, 1px hairline border (#dee1e6)

**Page Structure:**

1. **Header:** Minimal centered text logo "MSP" (Display md, weight 400).
2. **Registration Card:** Centered card containing:
   - Heading (weight 400): "Create Account"
   - Username input (12px radius, 48px height)
   - Email input (12px radius, 48px height)
   - Password input (12px radius, 48px height)
   - Confirm password input (12px radius, 48px height)
   - Primary blue CTA pill button ("Register")
3. **Footer:** "Already have an account? Sign in" text link in primary blue.
