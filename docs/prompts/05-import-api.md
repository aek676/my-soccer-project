A search and selection interface to import players from an external football API into the local database.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Mobile-responsive
- Theme: Institutional editorial interface
- Background: Soft Gray (#f7f7f7) to indicate a workspace area
- Primary Accent: Blue (#0052ff)
- Shapes: Pill checkboxes/buttons, circular avatars

**Page Structure:**
1. **Header:** Standard light navigation with text logo "MSP".
2. **Search Band:** White background card containing a large pill-shaped search input (e.g., "Search API Football for players...").
3. **Results Grid:** 
   - A 3-up grid of player cards (24px radius, white background).
   - Each card has a circular photo, name, team, and a pill-shaped toggle button (Select / Deselect).
4. **Floating Action Bar (Bottom):** Fixed at the bottom of the screen. Shows "X Players Selected" and a primary blue pill CTA "Import to Database".
